控制缓存版本号可以给 JC.pathPostfix 赋一个版本号值
<script> JC.pathPostfix = '20130718'; JC.use( 'Panel, Valid, plugins/jquery.form.js' ); </script> should be: <script src="libpath/comps/Panel/Panel.js?v=20130718"></script> <script src="libpath/comps/Valid/Valid.js?v=20130718"></script> <script src="libpath/plugins/jquery.form.js?v=20130718"></script>
